# Glide App Templates

Landing page for **glideapptemplates.com** showcasing LowCode Agency's Glide templates.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS 4
- TypeScript
- Lucide React icons

## Deployment

Deployed on Vercel - auto-deploys from GitHub `main` branch.
